Objective: Predict laptop prices based on their hardware specifications and brand attributes using regression modeling.

About the Dataset: A dataset of laptop listings with specs including Company, TypeName, Screen Size (Inches), Screen Resolution, CPU, RAM, Memory (HDD/SSD/Hybrid/Flash), GPU, Operating System, and Weight, with Price as the target variable.

Explored data with a price distribution histogram, a price frequency bar chart, and a price-by-OS boxplot
Built a Linear Regression model to predict laptop price from the processed features.
